M1卡中可存储1024Byte数据。
分为16个扇区，每个扇区为64Byte。
每个扇区有4个数据块，每个数据块为16Byte

## 扇区功能表

| Positioin   | 扇区ID | 数据块ID | 描述                        |
| ----------- | ------ | -------- | --------------------------- |
| <0, 0>      | 0      | 0        | Manufacturer Block 厂商数据 |
| <0-15, 1-2> | 0-15   | 1-2      | Data 常规数据               |
| <0-15, 3>   | 0-15   | 3        | Sector Traier 扇区数据      |

### Manufacturer block

厂商数据位于`<0, 0>`数据块
